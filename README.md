Overview
The Student Data Analyzer is a Python-based project that reads student data from a CSV file and performs data processing and analysis. It calculates total marks, averages, and generates visual graphs using Matplotlib.⚙️ Features
Read student data from CSV file
Calculate Total Marks
Calculate Average Marks
Data cleaning and formatting
Generate graphs using Matplotlib 📊
Simple and easy-to-understand code structure
Python 🐍
CSV Module
Matplotlib 📊
Basic Data Structures (lists, dictionaries)
